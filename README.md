# Royal Lens Studio - Premium Photography Landing Page

![Royal Lens Studio Banner](/images/preview.png)

## Overview

**Royal Lens Studio** is a high-performance, visually stunning landing page designed for a premium photography and cinematography studio. This project serves as a showcase of modern frontend development, featuring a responsive design, smooth animations, and a luxurious aesthetic tailored for high-end clients.

It demonstrates the ability to translate a premium brand identity into a functional, interactive web experience.

- **Premium Aesthetic**: A curated Gold & White color palette with elegant typography (Cormorant Garamond & Montserrat).

- **Smooth Animations**: Custom CSS animations and transitions for a polished user feel.
- **Interactive Gallery**: A masonry-style photo gallery with hover effects.
- **Video Showcase**: Integrated video section with autoplaying previews.
- **Contact Form**: A functional-looking contact form with validation states.
- **Modern Tech Stack**: Built with the latest React ecosystem tools.
